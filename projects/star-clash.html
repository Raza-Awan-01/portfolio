Star Clash

PC · Windows
Multiplayer 2D 3D Turn-Based Strategy
My Role in Project
  • Developed a multiplayer strategy game in Unity using Unity Netcode, supporting up to 8 players, leading the project as a team of 2 Developers.
  • Built a lobby system with public and private (password-protected) lobbies, quick match, and public lobby browsing.
  • Implemented dynamic map selection based on player count, with a random map chosen and synced across all players.
  • Designed 4 unique factions, each with distinct units, stats, and special abilities for varied strategic gameplay.
  • Created a data point-based win condition system with support for both solo and team-based play modes.
  • Implemented Fog of War to add tactical depth and limit enemy visibility.
  • Added configurable game settings including turn time limits, V-Sync, and screen mode adjustments.
  • Handled the complete final milestone independently after the team was reduced, delivering all remaining features solo.
