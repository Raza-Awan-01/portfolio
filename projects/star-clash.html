Star Clash

PC · Windows
Multiplayer 2D 3D Turn-Based Strategy
About the Game

Star Clash is a multiplayer turn-based strategy game built in Unity using Unity Netcode for real-time multiplayer synchronization. Players face off in tactical battles where every move counts.

The game blends both 2D and 3D visual elements to create a unique aesthetic, with strategic depth that rewards careful planning and outmaneuvering your opponent.
