Divine Chess

PC · Android
Multiplayer vs AI 2D Turn-Based Strategy Board Game
About the Game
  • Developed a multiplayer chess game in Unity as a solo developer, handling all aspects from gameplay programming to visual design.
  • Integrated Photon Fusion 2 for real-time multiplayer functionality across Quick Play and Ranked modes.
  • Implemented an ELO-based ranking system for Ranked mode with separate unranked Quick Play matches.
  • Designed and built a vs AI mode as a third gameplay option.
  • Created 5 unique playable characters (Deities), each with 2 custom special abilities that add strategic depth beyond traditional chess, such as the Ice Deity freezing adjacent enemy pieces for 2 turns.
  • Built an animated card UI system using DOTween to represent selected characters and surface ability buttons dynamically.
