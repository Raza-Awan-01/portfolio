Satoshi Runner

WebGL · Telegram
Online 3D Endless Runner Leaderboards Web3
My Role in Project
  • Developed a Web3 endless runner game in Unity, deployed as a WebGL game on Telegram with separate dev and production environments hosted on AWS.
  • Integrated Beamable backend for player authentication, global leaderboard rankings, and a league system (Bronze, Silver, etc.).
  • Implemented an NFT reward system, allowing players to earn NFTs through gameplay.
  • Built a custom automated build pipeline for dev/prod builds, handling Beamable realm config switching, and post-build modifications to index.html and CSS files, significantly reducing manual build time.
  • Integrated Telegram AdsGram for in-game advertisements and used JavaScript for Telegram bot setup and integration.
  • Added a daily login reward system to drive player retention.
  • Built and maintained an Android APK version for internal testing purposes.
