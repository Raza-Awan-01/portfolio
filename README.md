# Muhammad Raza — Game Developer Portfolio

Personal portfolio website showcasing my games and experience as a game developer.

🌐 **Live Site:** [raza-awan-01.github.io/portfolio](https://raza-awan-01.github.io/portfolio)

---

## 📁 Project Structure

```
portfolio/
├── index.html                  ← Main single-page portfolio
├── css/
│   ├── base.css                ← CSS variables, resets, shared utilities, footer
│   ├── nav.css                 ← Navbar & hamburger menu
│   ├── home.css                ← Hero section
│   ├── about.css               ← About section
│   ├── projects.css            ← Project cards & tags
│   ├── contact.css             ← Contact form & links
│   └── project-page.css        ← Individual project page styles
├── js/
│   ├── cursor.js               ← Custom animated cursor
│   ├── particles.js            ← Floating hero particles
│   ├── nav.js                  ← Hamburger toggle & active link highlight
│   ├── scroll.js               ← Scroll reveal animations
│   └── contact.js              ← Contact form submit handler
├── projects/
│   ├── star-clash.html         ← Star Clash project page
│   ├── divine-chess.html       ← Divine Chess project page
│   ├── gt-mega-ramp.html       ← GT Mega Ramp project page
│   └── satoshi-runner.html     ← Satoshi Runner project page
└── assets/
    ├── star-clash/             ← Icon & screenshots for Star Clash
    ├── divine-chess/           ← Icon & screenshots for Divine Chess
    ├── gt-mega-ramp/           ← Icon & screenshots for GT Mega Ramp
    └── satoshi-runner/         ← Icon & screenshots for Satoshi Runner
```
